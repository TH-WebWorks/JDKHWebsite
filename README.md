# JDKH Website

Public landing page for [jeffrey-didnt-kill-himself.com](https://jeffrey-didnt-kill-himself.com). Next.js, React, Tailwind CSS.

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm run start
```

## Deployment (Vercel)

1. Push this repo to GitHub (or connect your Git provider in Vercel).
2. In [Vercel](https://vercel.com), **Add New Project** and import the repository.
3. Leave **Framework Preset** as Next.js and **Root Directory** as `.`.
4. Deploy. Set the production domain to `jeffrey-didnt-kill-himself.com` in Project Settings → Domains.
